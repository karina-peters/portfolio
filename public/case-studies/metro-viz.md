# Sample Markdown File

Welcome to this sample Markdown file. Markdown uses plain text characters to apply formatting, making it simple to create documentation and web content.

---

## Headings

Headings are created using number signs (`#`). The number of signs corresponds to the heading level.

# Heading 1

## Heading 2

### Heading 3

#### Heading 4

##### Heading 5

###### Heading 6

## Text Formatting

You can format text as **bold**, _italic_, or **_both_**.

- This text is **bold** using double asterisks.
- This text is _italic_ using single asterisks.
- This text is **_bold and italic_**.

You can also use an inline code snippet, like `<div></div>`.

## Lists

Markdown supports both ordered and unordered lists.

### Unordered List

Use asterisks (`*`), plus signs (`+`), or hyphens (`-`) for unordered lists.

- First item in the list
- Second item in the list
- Third item in the list
  - You can create nested lists by indenting.

### Ordered List

Use numbers followed by periods for ordered lists.

1.  First ordered list item
2.  Second ordered list item
3.  Third ordered list item

## Code Blocks

Use three backticks (` ``` `) to create a fenced code block.

````markdown
```python
def hello_world():
  print("Hello, world!")
```
````

## Blockquotes

Use the email-style greater-than character (`>`) for blockquotes.

> Markdown is a text-to-HTML conversion tool for web writers.
>
> > Multiple paragraphs need to be prepended individually.

## Tables

Tables are supported using hyphens and vertical bars.

| Header 1     | Header 2     | Header 3     |
| :----------- | :----------- | :----------- |
| Row 1, Col 1 | Row 1, Col 2 | Row 1, Col 3 |
| Row 2, Col 1 | Row 2, Col 2 | Row 2, Col 3 |
